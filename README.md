This project is created as part of Rotaract Week learning activity to understand Git, GitHub, and Deployment concepts. It demonstrates how to initialize a Git repository, add files, commit changes, and push code to GitHub. The project also focuses on deploying applications using platforms like Heroku, Netlify, or Vercel.
Technologies Used:
Git
GitHub
HTML / CSS / JavaScript (if used)
Deployment Platforms (Netlify / Vercel / Heroku)
Purpose:
The main goal of this project is to learn version control and understand how to make an application live on a server.
